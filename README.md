# Likhamba Rongmei Portfolio

This repository contains my personal portfolio website, focused on cloud security and SOC analysis. It highlights the work I have been doing through hands-on labs, attack simulations, and detailed writeups based on real logs.

The site reflects how I approach learning security. I build environments, simulate attacks against them, and then analyze what actually happens at the log level. The goal is not just to understand tools, but to understand behavior.

The portfolio is built as a simple static site using HTML and CSS, and is hosted using GitHub Pages.

## What this site shows

Projects that involve AWS attack simulation and detection workflows
Hands-on SOC style investigations using CloudTrail and other logs
Writeups that explain how attacks look from a defender’s perspective
Current learning progress in cloud security and incident response

## Featured work

The main project showcased is an AWS Cloud SOC lab where multiple attack scenarios were simulated, including IAM privilege escalation, S3 exposure, and EC2 persistence. Each scenario was investigated manually through logs, with a focus on detection and response rather than automation.

## Why this exists

Most learning resources focus on theory or tools in isolation. This portfolio is built around practical understanding. The focus is on what actually happens during an attack and how it can be detected using real data.
